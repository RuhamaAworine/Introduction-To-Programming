1.Explain why we need long long int data type? 
We need to use long long int data type to store integer values that exceeds the 
limit on integer data type. int data type can store maximum(10^9) digit in memory ,
whereas long long int can store (10^18)digit in memory. so we need to use long long int 
to store values that are greater than (2^32)bits.
